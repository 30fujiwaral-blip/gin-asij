# GIN Club Website

A simple static website for the GIN (Global Issues Network) club at our school.

- Brand color: `#3b538b`
- Background: white
- Pages: `index.html`, `team.html`

## Preview locally
Open `index.html` directly in your browser, or run a local server, e.g.:

```bash
python3 -m http.server 8080
```

Then open http://localhost:8080 in your browser.
